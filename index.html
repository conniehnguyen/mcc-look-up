Credit Card Cashback Optimizer

Save up to 5 credit cards, look up by merchant name or enter a 4-digit POS code (MCC), and see which card gives you the best cashback return.

Your cards

Add up to 5 cards. Each card can have multiple categories, and each category can have its own cashback value.

Build a card by adding one or more category/reward pairs, then save the card.

Merchant or POS code lookup

Search by merchant name or enter a 4-digit code. Merchant lookup will try to find a known MCC first, and MCC lookup still works exactly as before.

Custom code-to-category overrides

If a merchant is categorized differently than expected, save your own category mapping for a 4-digit code.

This app stores your cards and overrides locally on your device.
